Bulk Password Best Practices

Store Securely

Import passwords into a password manager immediately. Never leave them in plain text files.

Never Email Plain Text

Don't send passwords via email. Use secure channels or password managers with sharing features.

Document Assignment

Keep a secure record of which password was assigned to which account or user.

Force Password Change

When distributing initial passwords, require users to change them on first login.

Frequently Asked Questions

Are the bulk passwords unique?

Yes. Each password is generated independently using cryptographically secure randomness. With the massive number of possible combinations, the chance of duplicates is astronomically low.

What's in the CSV download?

The CSV file contains a numbered list of passwords that can be opened in any spreadsheet application. You can add your own columns for usernames, accounts, or notes.

Is there a limit to how many I can generate?

The interface allows up to 100 passwords at once. For larger batches, you can generate multiple times. Since everything runs locally, there's no server limit.

Can I click individual passwords to copy?

Yes! Click any password in the list to copy just that one to your clipboard. You'll see a confirmation toast when copied.
